🌱 Micro Carbon Credit System

A full-stack web application that helps users convert electricity usage into green carbon credits.

🚀 Live Demo

👉 Frontend: "https://micro-carbon-credit.vercel.app"
👉 Backend API: "https://micro-carbon-credit.onrender.com/docs"

🛠️ Tech Stack
Frontend: React (Vite)
Backend: FastAPI
Database: PostgreSQL (Neon)
Hosting:
Frontend → Vercel
Backend → Render
✨ Features
User Registration & Login 🔐
Dashboard 📊
Carbon Credit Calculation 🌱
API Integration ⚡
📂 Project Structure
micro-carbon-credit/
│
├── backend/   # FastAPI backend
├── frontend/  # React frontend
